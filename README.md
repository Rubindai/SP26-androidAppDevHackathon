Big Red Path
Find your classes, stay on track, and organize your schedule all in one app.

Backend: https://github.com/RyanCheung555/hack-challenge-backend



Big Red Path is a comprehensive planning app for Cornellians studying CS, ECE, or ORIE in the 
school of Duffield Engineering. This app includes a checklist of major and college requirements
which would be updated in accordance to the classes the user adds to their schedule. To add classes,
the user can search up the classes they are taking, and add them to their schedule. The app includes
a Schedule screen, where the user will be able to add their own events along with viewing their 
classes.

1. Our app has at least one clear example of MVVM, particularly the Progress/Dashboard screen is 
separated into ProgressScreen.kt, ProgressViewModel.kt, and StudentRepository.kt
ProgressScreen.kt includes the UI of the screen, ProgressViewModel.kt includes the linkage between
UI and the model, and StudentRepository.kt includes the backend connection.
2. We have multiple screens, which includes the main screens called Dashboard, Search, and Schedule.
Other screens include the onboarding screens, when the user first opens up the app.
3. We used retrofit to connect the frontend with the backend.
